# Campus Lost & Found Management System

## ST. Peters College of Engineering and Technology (Autonomous)

A comprehensive web-based Lost & Found management system for institutional use.

### Features
- Student Portal: Report lost/found items, track claims
- Admin Dashboard: Manage items, students, claims, and messages
- AI-Powered Matching: Smart suggestions using similarity analysis
- Role-Based Access: Student, Admin, Super Admin roles

### Tech Stack
- Frontend: React + Tailwind CSS
- Backend: FastAPI + MongoDB
- Authentication: JWT-based

### Getting Started
1. Start backend: `cd backend && uvicorn server:app --reload`
2. Start frontend: `cd frontend && yarn start`

### Default Admin Credentials
- Username: superadmin
- Password: SuperAdmin@123
